CSS Variables

Allows you to store common (changable) values at one place, which can be changed by editing at only one place

  • Variables allow to have all dynamic values at one place
  • Allow us to change value of something throughout all styles sheet by changing at one place
  • Maintainability
  • Ease of development
  • Ease of understanding (due to good naming)
  • Theming
