Rems allows you to provide relative values without facing issues like em. However it doesn't mean em is bad or you should always use rem

When to use ems or rems?

Use rems for font-sizes, ems for margin/padding element sizing (height/width) and border radius, and px for border

  • This will make font sizes predictable
  • Keep margin, padding scaled w.r.t font-size
  • Pixels make sense for border, as they allow you to provide accurate values for border
