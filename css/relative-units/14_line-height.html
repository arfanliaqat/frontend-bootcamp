Line height

Line height allows you to change vertical spacing between lines of text elements

The technical word for CSS value which represents a distance measurements (such as 5px, 1em, 3rem) is Length. Note: Percentages may look like lengthy but technically they are not length. When an element has value defined using length (px, em, rem and so forth), its computed value is inherited by child elements. If line-height is specified with a unit, its value is calculated and that value is passed to children. This may lead to problems as shown here

  • Some properties allow unitless value
  • Some example are line-height, z-index, and font-weight
  • You can use unitless value 0 anywhere a length unit is required because in this case unit doesn't matter
  • A unitless 0 can only be used for length properties such as padding, border, and widths. It can not be used for angular or time based values
  • The line height property accept both units and unitless values
  • It is recommended that you use unitless values for line-height
